# 📘 Externship-Portfolio  
### *A Collection of Data Analytics Projects Completed Through Externships and Independent Study*

Welcome to my **Externship Portfolio**, a curated repository showcasing real-world data analytics, public health insights, visualization projects, and structured storytelling exercises completed as part of ongoing externships and professional development.

This portfolio follows a **clean, modular structure** where each project is self-contained inside the `/projects/` directory, with its own data, notebooks, visualizations, and documentation.
