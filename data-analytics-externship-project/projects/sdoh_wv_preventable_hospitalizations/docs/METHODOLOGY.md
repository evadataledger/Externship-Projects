# 🔬 METHODOLOGY.md  
### *Analytic Workflow for SDOH & Preventable Hospitalizations Project*

This document explains the analysis process used to complete the externship project.

---

# 🧭 1. Problem Definition
- Understand disparities in preventable hospitalizations across West Virginia counties.
- Identify which SDOH factors correlate with hospitalization rates.
- Present findings visually and narratively for stakeholder understanding.

---

# 🧼 2. Data Cleaning
- Loaded raw CHR&R data.
- Standardized county names and IDs.
- Removed irrelevant or duplicate features.
- Addressed missing and redlined data.
- Renamed inconsistent columns.
- Imputed missing values using median strategy.

---

# 🔗 3. Data Merging
- Merged selected CHR&R SDOH measures with supplemental data.
- Validated merge integrity (unique county keys).

---

# 📊 4. Exploratory Data Analysis (EDA)
- Summary statistics for each variable.
- Histograms to assess distribution.
- Scatterplots to compare SDOH variables with hospitalization rates.
- Correlation exploration to identify potential relationships.
- Outlier and data gap investigation.

---

# 🎨 5. Visualization & Dashboard Planning
- Selected core SDOH indicators for visual comparison.
- Created scatterplots, histograms, and maps.
- Structured a dashboard showing county-level metrics.
- Prepared visuals used in the project presentation.

---

# 📝 6. Storyboarding & Presentation
- Built a 3–5 minute narrative explaining:
  - The problem  
  - Key insights  
  - Risks & opportunity areas  
  - How SDOH influence hospitalization outcomes  

---

# 🎯 7. Final Deliverables
- Clean datasets  
- Jupyter/Colab notebook  
- Dashboard + interactive link  
- Presentation outline and slides  
- Project documentation (this folder)

---
